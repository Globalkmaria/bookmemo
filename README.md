## Installation

1. Git 클론 또는 다운받기

```bash
git clone https://github.com/Globalkmaria/bookmemo.git
```

2. 해당 문서 들어가기

```bash
cd bookmemo
```

3. 터미널에서 npm i 라이브러리 전체 설치

```bash
npm i
```

4. 터미널에서 npm start 로 서버 실행

```bash
npm start
```

- Visual Studio Code 에서 작성, 크롬에 최적화 되어있습니다.
